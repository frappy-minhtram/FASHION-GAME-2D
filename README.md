# FASHION-GAME-2D
Fashion Magic Battle kết hợp độc đáo giữa thể loại thay đồ truyền thống và cơ chế chiến đấu tính điểm theo lượt hệ RPG. Để chiến thắng, người chơi cần tính toán chiến thuật phối đồ dựa trên gợi ý từ NPC, nhằm khắc chế chính xác thuộc tính nguyên tố và phong cách thời trang của Boss.
